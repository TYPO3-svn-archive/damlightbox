Common markers that may be substituted by the TS configuration Markers: DAM_IMAGE - The current image from DAM DAM_TITLE - The title of the current DAM image DAM_COPYRIGHT - Copyright information for the current DAM image DAM_LOCATION - Location information for the current DAM image

IFRAME TEMPLATE

EXAMPLE

###DAM_TITLE###

###DAM_IMAGE###
###DAM_COPYRIGHT### ###DAM_LOCATION###
